# M2b-2 Implementation Plan — MemTrace + 记忆消融 + 叙事对抗 quorum 进阶

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Give the Playtest Agent a from-scratch, deterministic long-horizon memory (episodic trace + persistent state→action transition graph + reusable skill layer + verifier-weighted recall), prove its contribution with a memory ablation over the ≥20-chain corpus, and advance the Consistency assistant's 3-sample quorum to a perspective-diverse debate with a rebuttal round.

**Architecture:** `MemTrace` is a new, self-contained module in `agents/playtest/memory.py` implementing the protocol already consumed by `agent.py` (`record`/`recall`/`compact`/`reflect`). Recall is **deterministic arithmetic** (recency × structural-similarity × verifier-verdict) — zero model calls — so the memory structure and recall ranking are unit-testable without cassettes. The only LLM-touching addition is **recall injection** into the planner/executor user message and **compaction** via the Router; both are strictly gated so that `memory=None` produces **byte-identical requests** to M2b-1 (the M2b-1 cassettes must still replay). The memory ablation reuses the harness's existing `memory_factory` slot. Consistency's advance is additive: distinct perspective prompt variants + one rebuttal round, still `oracle_type="llm-assisted"` and human-confirmed.

**Tech Stack:** Python 3.12 via `uv`; pydantic v2 for any typed I/O; pytest + TDD; import-linter (7 contracts); ruff.

```python
# gameforge/agents/playtest/memory.py
"""MemTrace (M2b-2): from-scratch long-horizon memory for the Playtest Agent.

Design borrowed (not dependencies): TITAN (down-weight repeated no-progress) +
Voyager (reusable skill layer) + Generative-Agents (recency×relevance×verdict
recall) + HiAgent/AWM/MemGPT (compaction at task-step boundaries). Recall is
DETERMINISTIC arithmetic — no model call — so ranking is fully reproducible and
unit-testable. Only `compact` touches the Router (replayable), and only when a
MemTrace instance is actually attached (`memory is not None`); with no memory
the Playtest loop is byte-identical to M2b-1.
"""
from __future__ import annotations

from dataclasses import dataclass, field


@dataclass(frozen=True)
class Episode:
    state_abstract: str
    action: dict
    result: str
    state_hash: str = ""
    tick: int = -1
    step_index: int = 0
    verdict: float = 0.0  # verifier-conditioned weight delta (reflect/compact set it)


class MemTrace:
    def __init__(self) -> None:
        self.trace: list[Episode] = []

    def record(self, step: dict) -> None:
        self.trace.append(
            Episode(
                state_abstract=str(step["state"]),
                action=dict(step["action"]),
                result=str(step["result"]),
                state_hash=str(step.get("state_hash", "")),
                tick=int(step.get("tick", -1)),
                step_index=int(step.get("step_index", len(self.trace))),
            )
        )
```

```python
from collections import Counter

_NO_PROGRESS_RESULTS = frozenset({"blocked", "unreachable", "noop", "invalid", "already"})

    @staticmethod
    def action_key(action: dict) -> str:
        kind = str(action.get("kind", action.get("type", "?")))
        target = action.get("target") or action.get("target_id") or ""
        return f"{kind}:{target}"

    def no_progress_count(self, state_hash: str, action: dict) -> int:
        bucket = self.transitions.get(state_hash, {}).get(self.action_key(action))
        if bucket is None:
            return 0
        return sum(n for r, n in bucket.items() if r in _NO_PROGRESS_RESULTS)
```

### Task 3: verifier-weighted `recall` (recency × structural × verdict × optional embedding)

**Files:**
- Modify: `gameforge/agents/playtest/memory.py`
- Test: `tests/agents/test_memtrace.py`

**Interfaces:**
- Produces: `MemTrace.recall(state: str, task: object, k: int = 3) -> list[Episode]` and `MemTrace.recall_text(state, task, k=3) -> str | None` (formatted for prompt injection; `None` when no episodes). Scoring: `score = recency × structural_similarity × verdict_weight × embedding_similarity` where
  - `recency = 1 / (1 + age)`, `age = (len(trace)-1) - step_index`;
  - `structural_similarity = jaccard(tokens(state), tokens(ep.state_abstract))` (whitespace token sets; identical→1.0, disjoint→0.0);
  - `verdict_weight = 1.0 + ep.verdict − 0.5 × min(no_progress_count(ep.state_hash, ep.action), 4)` clamped to `[0.05, ∞)` — a state/action that historically stalled is suppressed (TITAN), a positively-reflected episode (`verdict>0`) is boosted;
  - `embedding_similarity` — **the injectable embedding term** `[user-decision 2026-07-08: ON as a real component, OFF in tests]`. `MemTrace(embedder: Embedder | None = None)`. When `embedder is None` (the default, all CI) this term is the neutral `1.0`, so recency/structural/verdict fully decide and recall stays deterministic + zero-model. When an `Embedder` is injected, it is `cosine(embedder.embed(state), embedder.embed(ep.state_abstract))` remapped to `[0.05, 1.0]`; the embedder is expected to be router-backed (RECORD/REPLAY) so embeddings are recorded and deterministic under REPLAY. Embeddings are cached per state-text within a `MemTrace` (same text → one embed call) to bound cost.
  Ties broken by later `step_index` first (recency), then insertion order — fully deterministic when `embedder is None`. The recency/structural/verdict math never calls a model; only the optional embedding term does.
- Also produces the `Embedder` Protocol (in `memory.py`): `class Embedder(Protocol): def embed(self, text: str) -> list[float]: ...`. This milestone ships NO concrete implementation wired into CI — the user supplies the embedding API + model later and it is injected via `MemTrace(embedder=...)` / `memory_factory` with zero further change to `recall`. A trivial deterministic fake embedder is used ONLY in unit tests that exercise the embedding branch (below).

  - `MemTrace.reflect(failed_trace: list[dict] | list[Episode], verdict: str) -> str` — deterministic: appends a verdict-conditioned reflection Episode (negative `verdict=-1.0` when `verdict` in the negative set `{"unreachable","abort_quest","stuck","fail"}`, else `0.0`) whose `state_hash` = the last failing episode's, so recall down-weights that path; returns the note string. TWO first-class `Compactor` implementations ship, behind a `Compactor` Protocol (`def compact(self, trace, verdicts, *, router=None) -> str`):
    - `DeterministicCompactor` — zero-model: extracts verified skills + a deterministic digest of the completed-step tail. Never touches `router`.
    - `LLMCompactor` — summarizes the tail via `call_model` (recorded/replayable through `router`); fail-closed to the deterministic digest on any parse/transport failure (never raises).
    `MemTrace(compactor: Compactor = DeterministicCompactor())` selects the strategy; `compact` delegates to it. BOTH are fail-closed. Task 8b runs the memory-on corpus under each strategy and reports which yields the better completion rate — the deterministic one keeps the whole layer zero-model (cleanest ablation), the LLM one may summarize better; the empirical comparison decides, quality-first. `compact` is only called by `agent.py` when `memory is not None`.
